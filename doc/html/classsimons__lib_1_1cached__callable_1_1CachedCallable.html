Detailed Description

template<typename result_type, typename mutex_type = DummyMutex>
class simons_lib::cached_callable::CachedCallable< result_type, mutex_type >

Simple cache for results returned by callable objects.

Template Parameters

mutex_typeInternally used mutex type (defaults to DummyMutex). If threadsafety is required supply a mutex of your choise.
