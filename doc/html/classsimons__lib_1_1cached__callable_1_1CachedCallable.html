Detailed Description

template<typename T, typename M = DummyMutex>
class simons_lib::cached_callable::CachedCallable< T, M >

Simple cache for results returned by callable objects.

Template Parameters
TThe cached result type.
MInternally used mutex type (defaults to DummyMutex). If thread safety is required supply a mutex of your choice.
