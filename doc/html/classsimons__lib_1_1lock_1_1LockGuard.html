Detailed Description

template<typename Mutex>
class simons_lib::lock::LockGuard< Mutex >

Simple re-implementation of std::lock_guard.

Note
This RAII lock guard is used throughout simons_lib with the intension, that the mutex header doesn't have to be included in cases where no threadsafety is required.
Template Parameters
Mutexthe mutex type to lock. Must implement at least BasicLockable.
